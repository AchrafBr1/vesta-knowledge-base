# TPLINK-144

## Adaptateur USB WiFi 150Mbps Wireless N Nano pour centrales VESTA-243

<figure><img src="../.gitbook/assets/image.png" alt="" width="93"><figcaption></figcaption></figure>

Design miniature élégant, si petit qu'une fois connecté, vous pouvez le laisser dans le port USB d'un ordinateur portable. Vitesse de transmission rapide jusqu'à 150 Mbps, idéale pour le streaming vidéo ou les appels Internet. Sécurité avancée : prend en charge 64/128 WEP, WPA, PA2/WPA-PSK/WPA2-PSK(TKIP/AES).

**Compatible avec VESTA-243 (ESGW)**
