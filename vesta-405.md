# VESTA-405

ADC Hybrid **(only compatible with Alarm.com)**

## 4G LTE Alarm.com hybrid alarm system

## Main features

* VESTA hybrid control panel with 4G LTE Alarm.com
* Incorporates via F1 868Mhz radio, WiFi, IP, Z-Wave and ZigBee
* 16 expandable wired zones
* Up to 160 zones in total (wired and wireless)
* 2 partitions and 80 users
* 1 PGM output (Siren)
* Up to 32 VESTA-114 keypads
* Configuration by smarthomesec APP
* Grade 3 certified.
